# Thinking In React

## 프로그램 설명

React 공식 문서의 [React로 사고하기](https://ko.react.dev/learn/thinking-in-react)를 학습하고 리팩터링한 프로젝트입니다.

## 기술 스택

- React 18.3.1
- Vite + SWC

## 실행 방법

1. 프로젝트 의존성 설치

```bash
pnpm install
```

2. 개발 서버 실행

```bash
pnpm dev
```

3. 브라우저에서 http://localhost:5173 접속
